# WhatsApp Chat FLirt Prediction and NLP Analysis App

A Streamlit web application to analyze WhatsApp chat exports. This app provides insights into chat activity, sentiment analysis, media usage, and generates a word cloud from chat messages.

## Features

- **Date-wise message count**: Visualize chat activity over time with counts per user.
- **Sentiment analysis**: Classify messages as positive or negative using NLTK VADER.
- **Flirt encounters**: Count potential flirt messages per user (based on keywords).
- **Media encounters**: Summarize media shared, deleted messages, missed audio and video calls.
- **Word Cloud**: Generate a word cloud from all chat messages.
- **Topic Modeling**: Discover hidden topics or themes in the chat using NLP techniques.
